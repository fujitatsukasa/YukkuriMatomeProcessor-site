# Task: SaaS Pricing UI Ultimate Refinement

## 概要
ランディングページの「料金プラン」セクションについて、妥協のない最高峰のSaaSデザインへと昇華させる。
これまでの度重なる修正漏れや先祖返り（退行）を防ぐため、確定した仕様を本ドキュメントにて厳格に管理する。

## 絶対仕様（Must Haves）
1. **チェックマークの横位置（X軸）の完全統一**
   - 現在、プランのテキスト長に依存してチェックリスト全体が中央揃えっぽくなり、チェックマークの始まる「横位置」が各カードでガタガタになっている。
   - すべてのカードにおいて、チェックマークが整然と左端（ただしカードの縁からは適切な余白を空けた位置）から横一直線に揃ってスタートするように `align-items` やマージンを調整すること。

2. **不要なリンク（CTA）の一切の排除**
   - 「Freeで始める」「Standardプランを選択」「Proプランで最速自動化 →」などの余計なボタン・リンクは表示しない。
   - このシステムはソフト内でしか課金機能・ログインを持たないため、LPには単なる「比較表」としての最高の見栄えだけがあればよい。

3. **バッジ・光彩の描画（オーバーフロー対策）**
   - カード要素の `overflow: hidden`等によって、カード上部の「個人クリエイター向け」「THE ULTIMATE」などのバッジや、Proプランの強力な光彩効果が切断（クリッピング）されないこと。

4. **1行ごとの隙間（Gap）の最適化**
   - チェックリスト間の余白が間延びしないよう、0.5rem程度にタイトに引き締める。縦にも横にも揃った、緊張感のあるプロフェッショナルなデザインとする。

5. **カスデザインからの脱却（最高峰のアニメーション）**
   - Reactのランタイムエラー（クラッシュ）を引き起こすような破壊的なFramer Motionの干渉は避け、堅牢な CSS を用いて以下を実現する。
     - チェック行へのホバーで、文字が白く発光しながら右へ滑らかにスライドする。
     - ホバー中、チェックアイコン自体がスピン（回転）しながら少し拡大する。
     - Proプランの価格「¥10,000」が脈打つように継続的に光を放つ（Pulse Glow）。
