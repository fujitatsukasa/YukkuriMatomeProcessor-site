# Implementation Plan: SaaS Pricing UI

## 目的
1. 「チェックマークの始まる横位置（X軸）がバラバラになっている」という非常にアマチュアなレイアウトの崩れを完全に修正する。
2. これまでに適用した「バッジの見切れ防止」「不要なCTAの削除」「リッチなホバーアニメーション」を絶対に壊さずに維持する。

## 修正策（チェックマークの左揃え）
### 現在の問題点
`.premium-pricing-features` やその親要素で `items-center` や `text-align: center` がかかっているか、マージンが `0 auto` のように設定されてしまっていることが原因。

### 解決手順 (react-site.css / home-page.tsx)
1. `.premium-pricing-features` (チェックリストのコンテナ) に対して、横幅を持たせつつ `display: flex; flex-direction: column; align-items: flex-start;` を適用する。
2. そのままだとカードの左端ギリギリに寄ってしまうため、適切な `padding`（例: `padding: 0 1rem;` やマージン）を設けて、カード全体としては中央のバランスを保ちながらも、**文字列やチェックマークのスタート地点は完璧な垂直線を引くように揃える**。
3. Proプランだけでなく、FreeやStandardプランにおいても同じクラス定義（または同じパディング量）を適用し、横方向のピクセルずれを0にする。

## 品質保証
修正完了後、Gitへのコミットを行う前に、開発環境（Vite）のエラーが出ていないかを確認し、必ず画面の目視確認・リグレッションテストにて「過去の仕様が壊れていないこと」と「本当に横一列に揃っていること」を検証する。
