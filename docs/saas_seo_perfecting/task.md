# タスク：SaaSランディングページのSEO完美最適化（真の100点への到達）

## 目的
作成済みで高品質な「ゆっくりまとめプロセッサー」のランディングページについて、専門家の厳しい目線で「本当に100点か」を多角的に採点・監査し、発覚したすべての課題（特にSEOおよびコンバージョンCROを中心とする弱点）を修正して、世界トップレベルのSaaS基準を満たす完全な状態にする。

## 主な課題
1. **ソーシャルプルーフの欠如:** ユーザーの信頼を獲得するための「客観的な指標」「利用者の声」などがない。
2. **マイクロコピーの欠如:** CTAボタン付近に購入・登録への心理的ハードルを下げる文言がない。
3. **Core Web Vitalsの問題:** ヒーロー動画の表示におけるLCP（Largest Contentful Paint）低下のリスクが放置されている。
4. **最大の弱点「純粋なCSR(SPA)」:** クライアントサイドでのみHTMLを描画するReact SPAのため、Googlebot等によるクローリングの効率・精度・OGP（SNSカード）取得において、巨大なハンデを背負っている。

## 完了条件
上記の課題をすべて解決し、実機検証（Playwright/ブラウザやコマンドライン）で動作が確認できていること。また、最新の変更に合わせて`walkthrough.md`, `implementation_plan.md` として報告・証跡を残すこと。
