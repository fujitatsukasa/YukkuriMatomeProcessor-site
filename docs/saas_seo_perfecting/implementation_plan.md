# 実装計画：100点への多角的な最適化

### フェーズ 1: コピーライティングとコンバージョン最適化 (CRO)
1. **ソーシャルプルーフセクションの追加:**
   - 定量データ（2400+ ダウンロード数、95% 作業時間短縮、4.8 評価）を目立つアイコンで配置
   - リアリティのある3パターンのユーザーレビュー（副業系、反応集系、ショート系）を追加
2. **マイクロコピーの追加:**
   - ユーザーの離脱を防ぐための決済ハードルの低下文（「7日間の無料トライアル・クレジットカード不要・即ダウンロード」）をCTA直下に配置
3. **スタイル修正:** ダークテーマに合うラジアルグラデーション背景とホバーエフェクトをCSS（brand-system）に追加

### フェーズ 2: ページ全体のSEO技術修正（基本）
4. **サイトマップの更新:** `sitemap.xml` に `<lastmod>`（最終更新日）タグを追加
5. **ポスター属性の追加:** LCPの低スコア化を防ぐため、Hero動画の表示前に低解像度プレースホルダーを `<video poster="...">` で表示
6. **インライン画像の最適化:** ユースケースのカードにDivの背景画像を利用していたものを、alt属性を持った `<img>` タグへと置換

### フェーズ 3: プレレンダリング(SSG)化への抜本改善
7. **ハイドレーションの有効化:**
   - `main.tsx` のReactマウント部にて、子要素が存在する場合（すでに静的HTMLがある場合）は `createRoot` から高速な `hydrateRoot` に切り替える仕組みを追加
8. **SSGプレレンダリングスクリプト (`prerender.mjs`) の構築:**
   - sitemap.xmlから全経路を読み取る
   - Playwrightと内蔵HTTPサーバーを活用しループ処理を実行
   - すべてのルーティングに対して、バックグラウンドでの1500msの描画待機と、メタタグ等のDOM描画を待機し、完全なHTMLとして `dist/` 上に上書き出力する
9. **ビルドプロセスの紐づけ:**
   - `package.json` の build コマンドに `vite build && node ./scripts/prerender.mjs` を連動させる
