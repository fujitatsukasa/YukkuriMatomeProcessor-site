# ChatGPT相談用パック

このZIPは、`ゆっくりまとめプロセッサー` Home LP v2を100点に近づけるために、ChatGPTへ渡す材料をまとめたものです。

## 使い方

1. ChatGPTに `chatgpt-lp-100-review-20260622.zip` を添付する。
2. `PROMPT_FOR_CHATGPT.md` の内容をそのまま貼る。
3. ChatGPTがZIP内を読めない場合は、`materials/home-lp-v2-current-20260622/` を展開して、以下を優先添付する。
   - `00-codex-evaluation.md`
   - `01-current-home-copy.md`
   - `02-implementation-status.md`
   - `03-verification-report.md`
   - `04-known-gaps-and-next-actions.md`
   - `spec/ゆっくりまとめプロセッサー_Home_LP全面改修仕様書_v2.0_販売コピー確定版.md`
   - `screenshots/`
   - `source/home-content.ts`
   - `source/home-page.tsx`
   - `source/home-page.css`

## 同梱内容

- `PROMPT_FOR_CHATGPT.md`
  - ChatGPTへ貼る本命プロンプト。
- `PROMPT_SHORT.md`
  - 短く聞きたい場合の簡易版プロンプト。
- `materials/home-lp-v2-current-20260622/`
  - 現在のLP実装、コピー、仕様書、素材、スクリーンショット、検証レポート。
- `materials/home-lp-v2-current-20260622.zip`
  - 上記現状パックのZIP版。
- `materials/optional-previous-home-lp-brief-20260621.zip`
  - 任意参照用の前回ブリーフ。基本はv2現状パックだけで判断可能。

## 相談のゴール

- 現状のLPを、購入前ユーザーの不安解消、販売コピー、UI/UX、信頼性、アクセシビリティ、レスポンシブ、SEO、実装現実性の観点で100点に近づける。
- 嘘の実績、誇大表現、架空レビュー、収益保証、完全自動生成のような誤認表現は入れない。
- 実装担当者がそのまま作業に移れるレベルの改善仕様を得る。
