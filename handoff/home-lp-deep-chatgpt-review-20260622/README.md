# Home LP 詳細再レビュー用パック

作成日: 2026-06-22

このZIPは、ゆっくりまとめプロセッサー Home LPをChatGPTに再レビューしてもらうための詳細資料です。

目的は、前回の「64 / 100」レビュー後に行った安全修正を踏まえ、100点へ近づけるために必要な追加情報、素材、コピー、実装タスクを洗い出すことです。

## まず読むもの

1. `00-start-here/READING_ORDER.md`
2. `01-prompts/DETAILED_PROMPT_FOR_CHATGPT.md`
3. `00-start-here/APP_FACTS_DIGEST.md`
4. `02-current-lp/CURRENT_LP_STATE_SUMMARY.md`
5. `03-review-history/chatgpt-64-review.txt`

## このパックの見方

- `01-prompts/` は、そのままChatGPTへ貼るプロンプトです。
- `02-current-lp/` は、現在のHome LPのソース、表示テキスト、構造化データ、スクショです。
- `03-review-history/` は、前回64点レビューと、その後の修正概要です。
- `04-app-fact-evidence/` は、アプリ側資料から見える根拠です。サイト改善時はここを最優先で照合してください。
- `05-assets/` は、現在LPで使っている主要アセットです。
- `06-screenshots/` は、今回撮影したローカルプレビューの実画面です。
- `07-reference-packs/` は、過去の相談用ZIPです。
- `08-next-codex-brief/` は、ChatGPTの回答後にCodexへ渡す実装依頼テンプレートです。
