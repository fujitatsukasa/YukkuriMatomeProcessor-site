# WebView2 Runtime 問題

このページでは、WebView2 Runtime が原因で YMP の画面が表示されない場合の確認手順を説明します。

## WebView2 Runtime とは

YMP の画面を表示するために必要な Microsoft の部品です。Windows 10/11 でも、環境によっては入っていない、古い、壊れている、セキュリティソフトに止められていることがあります。

## よくある症状

- 起動しない
- 白い画面になる
- ウィンドウは出るが UI が表示されない
- 起動直後に閉じる
- WebView2 に関する警告が出る

## 確認手順

1. YMP を終了します。
2. Microsoft Edge WebView2 Runtime が入っているか確認します。
3. 入っていない場合は、Microsoft 公式ページから Runtime をインストールします。
4. PC を再起動できる場合は再起動します。
5. YMP を起動し直します。
6. まだ起動できない場合は、診断画面で確認情報を作ります。

公式ダウンロードページ:

https://developer.microsoft.com/microsoft-edge/webview2/

## それでも直らないとき

### インストール後も白い画面

PC 再起動、YMP の起動し直しを試します。

### Runtime のインストールに失敗する

Windows Update、管理者権限、セキュリティソフトを確認します。

### 会社や学校の PC で入れられない

管理者に WebView2 Runtime が必要なことを伝えます。

### 更新後に急に出た

更新前後の版と、表示されたエラー文を控えます。

診断レポートには、WebView2 の有無や版などの確認情報だけを入れます。パスワード、認証情報、API キー、支払い情報、台本本文、プロジェクト本文は送らないでください。
