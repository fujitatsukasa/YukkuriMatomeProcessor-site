# 初回セットアップ

このページでは、YMP を初めて使うときに確認することを説明します。

## できること

インストール、起動、ログイン、Pro 状態、保存先、最初の動作確認までを順番に確認できます。初回だけでなく、PC を変えたときや入れ直したときにも同じ順番で見直せます。

## 始める前に

- Windows 10/11 x64 の PC を使っている
- Microsoft Edge WebView2 Runtime が入っている
- インターネットにつながっている
- ログインに使うアカウント情報が分かっている
- YMM4 を使う場合は、YMM4 の場所が分かっている
- 未署名配布では、Windows SmartScreen や発行元不明の警告が表示される場合がある

## 手順

1. `YukkuriMatomeProcessor-<version>-win-x64-standard-Setup.exe` を実行します。
2. インストールが終わったら Launcher を起動します。
3. WebView2 Runtime の警告が出た場合は、案内に従って Runtime を入れてから起動し直します。
4. ログイン状態を確認します。
5. Pro 状態を確認します。
6. 更新確認に失敗や警告が出ていないか見ます。
7. Main を起動します。
8. ツールからパス設定を開き、YMM4、素材、出力先の場所を確認します。
9. 台本取得で少ない件数の素材を取得します。
10. 台本編集画面で、取り込んだ内容が表示されるか確認します。

## 初回に完了していればよい状態

- Launcher が開く
- Main が開く
- ログイン状態が分かる
- Free / Pro 状態が分かる
- パス設定を確認できる
- 台本取得から台本編集まで一度試せる

## うまくいかないとき

### 起動しない

WebView2 Runtime、Windows の警告、セキュリティソフトの隔離を確認します。

### ログインできない

アカウント情報とインターネット接続を確認します。

### Pro 表示が違う

ログイン中のアカウントと購入に使ったアカウントを確認します。

### Main が開かない

起動画面に出ているエラー文を控えます。

### 台本取得できない

URL、取得件数、相手ページが開けるかを確認します。

解決しない場合は、診断画面で確認情報を作り、サポート窓口へ送ってください。パスワード、認証情報、API キー、支払い情報、台本本文、プロジェクト本文は送らないでください。
