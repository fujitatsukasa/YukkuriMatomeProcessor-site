# Agent Rules
- Create new Python files under `python/`.
- Create new QML files under `qml/`.
- Always respond in Japanese.
- Use custom log for logging; do not use `print`.
- Do not use JavaScript.
- UI-related text must be Japanese or English only.
- Avoid any character garbling (e.g., `???`).
- Treat text encoding as UTF-8 and explicitly specify it for input/output.
- `launcher.py` と `main.py` の処理は相互に依存しない（明示的に指定がある場合を除く）。
- `launcher` フォルダ内に `python/` と `qml/` を作成し、`launcher.py` 側の新規実装はそれぞれに配置する。
